🪝 Drone Cable Routing Accessory
Integrated Project II – Team 6
Universitat de Vic – Universitat Central de Catalunya. Academic Year 2025–2026.

📌 Project Overview
This project focuses on the design and assembly of an attachable hook for a drone specifically engineered for cable routing tasks. The system is remotely controlled, allowing for efficient cable deployment operations in complex or hard-to-reach environments.

🎯 Objectives
Main Objective
To design and assemble a remote-controlled hook attachment for drones.

Specific Objectives
Develop a pulley system for cable management.

Integrate safety features to ensure reliable device operation.

Establish stable RF (Radio Frequency) communication for remote control.

Validate the system through integration tests and optimization.

🏗 System Architecture
The development is divided into three fundamental technical pillars:

⚙️ Mechanical System: Includes the mechanical design of the hook and pulley, as well as the selection of materials and structural components.

⚡ Electronics System: Comprises the design of the control electronics, motor management, and overall system integration.

🧠 Software & Communications: Focused on firmware development and the configuration of the RF communication protocol.
